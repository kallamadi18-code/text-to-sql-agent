# Text-to-SQL LLM Agent

## 📋 Overview
An intelligent agent that converts natural language questions into SQL queries, allowing users to interact with databases using plain English.

## 🚀 Features
- Convert natural language to SQL queries
- Interactive web interface
- Sample database with employees, departments, and salaries
- 7 predefined query templates

## 🛠️ Technologies Used
- Python
- Streamlit
- SQLite
- Pandas

## 📊 Sample Queries
- "Show all employees"
- "List employees in Engineering department"
- "Show employees with salary > 50000"
- "Count employees by department"
- "Show average salary by department"
